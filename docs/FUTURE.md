# Ideas for later

Erga is still early. Some useful next steps are:

- make setup easier;
- let users review and approve imported experience from the CLI;
- improve application and recruiting-email history;
- support more résumé layouts;
- improve job-to-résumé matching; and
- eventually add a simple local interface.

Erga is not meant to automatically submit applications, message recruiters, or make up résumé
content.
